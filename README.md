# UAS Mobile Programming - EndemikDB

Aplikasi **EndemikDB** adalah aplikasi informasi hewan dan tumbuhan endemik Indonesia yang dibangun untuk memenuhi tugas akhir (UAS) mata kuliah Mobile Programming.

## Fitur Utama
- **Splash Screen**: Menampilkan logo aplikasi saat pertama kali dibuka.
- **Home**: Daftar hewan dan tumbuhan endemik (Grid View).
- **Search**: Mencari data berdasarkan nama.
- **Detail**: Informasi lengkap mengenai spesies tertentu.
- **Favorite**: Menyimpan data ke dalam database lokal (ROOM).
- **Dark & Light Mode**: Mendukung tema sistem secara otomatis.
- **Profil Pengembang**: Informasi mengenai pembuat aplikasi.
